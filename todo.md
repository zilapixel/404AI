# 404AI Website - Development Plan

## Design System

### Color Palette
- Background Primary: `#0A0A0B` (near-black)
- Background Secondary: `#121214` (dark gray)
- Surface/Card: `#18181B` with `rgba(255,255,255,0.04)` overlay
- Border Glass: `rgba(255,255,255,0.08)`
- Text Primary: `#FFFFFF`
- Text Secondary: `#A1A1AA` (muted gray)
- Accent Red: `#EF4444` (subtle red accent)
- Accent Red Glow: `rgba(239, 68, 68, 0.25)`

### Typography
- Primary Font: Inter (400/500/600/700/800/900)
- Display: tight letter-spacing (-0.03em to -0.05em) for hero

### Design References
- Stripe.com, Apple.com, Linear.app
